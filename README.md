# 🔐 File Integrity & Hash Checker

A Python-based cybersecurity tool that generates **MD5**, **SHA-1**, and **SHA-256** hashes to verify file integrity and support basic digital forensics and malware analysis workflows.

## 📌 Overview

The **File Integrity & Hash Checker** is a lightweight command-line tool developed in Python that calculates cryptographic hash values for files. It can be used to verify file integrity, perform basic digital forensic analysis, and support malware investigation workflows.

This project was created to strengthen practical cybersecurity and Python programming skills while exploring concepts used in real-world security operations.

---

## ✨ Features

* ✅ Generate **MD5** hashes
* ✅ Generate **SHA-1** hashes
* ✅ Generate **SHA-256** hashes
* ✅ Verify file existence before processing
* ✅ Efficient chunk-based file reading
* ✅ User-friendly command-line interface
* ✅ Supports files of various sizes

## 🚀 Getting Started

### Prerequisites

* Python 3.x installed on your system

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/file-integrity-hash-checker.git
cd file-integrity-hash-checker
```

### Run the Application

```bash
python hash_checker.py
```
